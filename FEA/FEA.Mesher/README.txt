The FEA Mesher module will translate iges and stl files to Finite Element Meshes using GPU/ MPI acceleration

Brainstorm ideas include surface node insertion followed by body node insertion and mesh smoothing 

Paramount to this is the NURBS surface which allows for accurate modelling of a wide array of surface types
